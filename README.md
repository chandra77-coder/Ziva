# Ziva

Ziva is a premium offline Android PDF and image toolkit.

Repository: `chandra77-coder/Ziva`
Package ID: `com.zivaapp.ziva`
Version: `1.0.0`

## Status

The app source is being uploaded and improved.

## Download APK

After GitHub Actions finishes:

1. Open the Actions tab.
2. Click the latest Build Ziva APK run.
3. Download the APK artifact.
4. Extract the ZIP and install the APK on Android.

## Privacy

Ziva is designed to work offline. No login, no ads, no analytics, no cloud upload.
